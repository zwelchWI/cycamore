Detailed Description

The GrowthRegion class inherits from the RegionModel class and is dynamically loaded by the Model class when requested.

This region determines if there is a need to meet a certain capacity (as defined via input) at each time step. If there is such a need, the region will determine how many of each facility type are required and then determine, facility by facility, which of its institutions are available to build each facility.
